How search works
Wafrn connects with the fediverse and bluesky. You can search tags "tumblr style" but for better results we recommend to use them on "twitter style" for fandoms
You cansearch public posts that wafrn has already seen. You can make we get more posts of a type by following a tag!
You can search for users. Search will return users by url. You can search for fedi users but you need to use the full @: as example, @foone@digipress.club
